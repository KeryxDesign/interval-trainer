# Interval Trainer

Simon cromatico per ear training a intervalli. Ogni semitono dalla tonica ha un colore dello spettro visivo, un simbolo e un concetto (il "viaggio cromatico": casa, fiamma, sole, stella, foglia, albero, onda, cielo, ancora, luna, fiore, fulmine). Il gioco genera una melodia casuale sui gradi della scala scelta, partendo sempre dalla tonica, e va ripetuta toccando i tasti.

Il sistema visivo è relativo (movable-do): cambiare tonica cambia solo l'audio, mai colori e simboli. È l'allenamento dell'orecchio relativo, non delle note assolute.

**Live**: https://keryxdesign.github.io/interval-trainer/

Vanilla JS, Web Audio API, nessuna dipendenza di build. Icone: Tabler Icons webfont via CDN.
